This project is a Discord bot specifically for use with the Discord server. It provides numerous utilities and other tools to help keep the server running like a well-oiled machine.

Цей проект є ботом Discord спеціально для використання з сервером Discord. Він надає численні утиліти та інші інструменти, які допомагають серверу працювати як добре налагоджена машина.
